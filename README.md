# MUST STEPS TO DO :

- STAR and FORK the repo,
- if adding a program ADD it to its relevant category,
- if adding a project then create a folder and add everything there,

## steps to push :

- commit with a good message
- push to your branch or MAIN branch
- Create PUll request
